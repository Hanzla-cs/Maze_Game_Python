# Maze Game Python

A multi-level maze game built with Python and Pygame. The player navigates through three custom-designed levels, avoids walls and enemies, and reaches the winning flag to progress.

## Features

- Three playable maze levels
- Pixel-based collision detection
- Enemy and obstacle collision logic
- Start, win, and loss screens
- Custom maps, player sprites, enemies, and flag assets
- Organized project structure with separate source code and assets

## Tech Stack

- Python
- Pygame
